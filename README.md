# Final Project WebDev RISTEK 2025 Tugas Tracker
## [cite_start][Tugas Tracker] adalah aplikasi web yang dikembangkan sebagai Final Project Open Class RISTEK Fasilkom UI 2025[cite: 1, 4]. [cite_start]Aplikasi ini bertujuan untuk memecahkan masalah mahasiswa yang sering kewalahan dalam mengelola tugas.

Website ini berfungsi sebagai sistem terpusat untuk melacak semua tugas, memungkinkan pengguna untuk:
* [cite_start]Mengelola mata kuliah (CRUD)
* [cite_start]Mengelola tugas yang terikat pada mata kuliah (CRUD)
* [cite_start]Melacak progres setiap tugas dengan status ('Belum Dikerjakan', 'Sedang Dikerjakan', 'Selesai')
